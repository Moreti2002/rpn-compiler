# Teste Completo: Operações e Armazenamento com Debug Print
# Este teste demonstra operações aritméticas e persistência de variáveis na SRAM
# Os resultados serão impressos via UART para validação

# Inicializar variáveis base
(10 A)
(20 B)
(5 C)

# Operações básicas - A + B = 30
(A B +)

# Subtração - A - B = -10 (será 246 em unsigned 8-bit)
(A B -)

# Multiplicação - B * C = 100
(B C *)

# Soma - A + C = 15
(A C +)

# Operações aninhadas - (A+B) + (C*2) = 30 + 10 = 40
((A B +) (C 2 *) +)
